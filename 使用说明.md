## 🔧 开发版本（需要API密钥）

如果您想使用完整的AI功能，请：

### 1. 配置API密钥
编辑 `.env` 文件：
```bash
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

### 2. 启动开发服务器
```bash
# 安装依赖（如果还没有）
npm install

# 启动开发服务器
npm run dev

# 或构建生产版本
npm run build
npm run preview
```
